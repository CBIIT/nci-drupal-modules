INTRODUCTION
============

Navbar Flush Cache is a module which adds functionality to the Navbar module
allowing users to flush the cache from the menu. The module is based on the
functionality provided by the Administration Menu module which also allows users
to choose which part of the cache to flush.


INSTALLATION
============

 1. Install and configure the dependencies. Check their README files for
    details.
 2. Install and enable the Navbar Flush Cache module.
 3. Define permissions on the admin/people/permissions page.


DEPENDENCIES
